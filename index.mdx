  <h2 className="text-3xl font-semibold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get your portfolio up and running in minutes</p>
  
  <Steps>
    <Step title="Clone the Repository">
      Download the portfolio template to your local machine.
      
      ```bash Clone repository
      git clone https://github.com/luisllatas-dev/portafolio-programador-web-junior.git
      cd portafolio-programador-web-junior
      ```
    </Step>
    
    <Step title="Open in Browser">
      Simply open the `index.html` file in your web browser to view the portfolio.
      
      ```bash Open with live server
      # Using Python
      python -m http.server 8000
      
      # Using Node.js
      npx serve
      ```
      
      <Note>
        No build process required! This is a static HTML/CSS website that works out of the box.
      </Note>
    </Step>
    
    <Step title="Customize Content">
      Edit the HTML and CSS files to personalize with your own information, skills, and projects.
      
      ```html index.html
      <h2>Desarrollador Full Stack</h2>
      <p>Soy un desarrollador web con experiencia...</p>
      ```
    </Step>
  </Steps>
</div>
